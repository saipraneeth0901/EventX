# 🎉 EventX – Event Design & Management Website

**EventX** is a fully responsive front-end website built for an **event planning and management company**.  
It allows users to explore event services, view galleries, and connect through an integrated contact form — designed for a premium and interactive experience.

---

## 🚀 Features

- 🏠 **Home Section:** Engaging hero carousel with call-to-action buttons.  
- 🧠 **About Section:** Brief overview of the company and its purpose.  
- 🛠️ **Services Section:** Dynamic cards for wedding, corporate, birthday, and cultural events.  
- 🖼️ **Gallery:** Interactive photo lightbox showcasing past events.  
- 📞 **Contact Form:** User-friendly form with real-time validation and submission feedback.  
- 🎨 **Responsive Design:** Optimized for desktop, tablet, and mobile using Bootstrap 5.  
- ✨ **Animations:** Implemented using **AOS** and **Lightbox.js** for smooth transitions.

---

## 🧩 Tech Stack

| Layer | Technologies Used |
|-------|--------------------|
| **Frontend** | HTML5, CSS3, JavaScript (ES6), Bootstrap 5 |
| **Libraries/Plugins** | AOS (Animations), Lightbox.js, Font Awesome |
| **Design Tools** | Google Fonts (Poppins), Unsplash (Images) |
| **Editor** | Visual Studio Code |
